Information concernant le visiteur

SushiFast est actuellement un site de commande en ligne dont le paiment se fait en physique dans une de nos boutiques partenaires. Le but est de pouvoir proposer une expérience en ligne à nos clients qui une fois sur place dans une de nos boutiques partenaires auront juste à payer et prendre leur commande.

Un site web sécurisée

SushiFast est un site internet utilisant le protocole https et hébergé chez GitHub via leur modules "pages"

SushiFast est un site utilisant la dernière version d'Angular et régulièrement mis à jour afin de proposer une expérience sécurisée
